# lilBrowser

> A Quasar project

## Build Setup

``` bash
# install dependencies
$ npm install

# install quasar-cli
$ npm install -g quasar-cli

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
